Create a Chord Diagram and Extract relevant sections from PDFs via a DOI cross-reference search.

The code bit are in the Create Chord folder, and the DOI Search & Extract folder.
The required files are the TXT and CSV.

The only thing missing are the PDFs. Simply gather your PDFs, then change the keywords and categories within the Create Chord python script to suit your needs.
The Citation Export file can be created by copying directly from the html file, so all you will neeed to do is modfiy the search words in the DOI Seach & Extract python script.

--> To view the chord diagram use Firefox. Chromium (Chrome & Edge) do not play well.
